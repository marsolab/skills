# Skills

This repository contains the skills for the AI coding agents.

## Skills list

- [Go](go/SKILL.md)
- [Copywriting](copy/SKILL.md)
- [System Architect](system-architect/SKILL.md)

## How to add a new skill

1. Create a new folder in the `skills` directory.
2. Create new files in the folder called `SKILL.md`, `CHANGELOG.md`.
3. Add the skill to the `skills` list in the `README.md` file.
